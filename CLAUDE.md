# Git workflow

Даалгавар/өөрчлөлт дуусах бүрд зөвшөөрөл асуухгүйгээр:
1. Өөрчлөлтүүдийг эмх цэгцтэй, логик бүлгээр тусдаа commit болгож хийнэ (нэг том холимог commit биш).
2. Commit message-ийг монгол хэлээр, ердийн өгүүлбэрээр бичнэ (угтвар/tag хэрэггүй) — жишээ нь энэ repo-гийн одоо байгаа commit түүх шиг.
3. Co-Authored-By: Claude мөр нэмэхгүй.
4. Бүх commit хийж дууссаны дараа шууд `git push origin main` хийнэ — push хийхээсээ өмнө асуухгүй.

Энэ нь зөвхөн энгийн commit+push-д хамаарна. Force push, `git reset --hard`, branch устгах гэх мэт бусад буцаах боломжгүй/аюултай git үйлдэлд энэ зөвшөөрөл хамаарахгүй — тэдгээрт хэвээр зөвшөөрөл асуух ёстой.
